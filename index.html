How to use this editor

  • Switch between ESX/QBCore using the tabs above to configure vehicles for your framework
  • Add categories to organize your vehicles (e.g., "super", "sedan", "motorcycle")
  • Add vehicles with model name, display name, price, and other attributes
  • Import existing configurations to edit them visually
  • Export to Lua/SQL when finished to get the correctly formatted code for your framework
  • Search and filter vehicles using the search bar and category tabs
  • Duplicate vehicles to quickly create similar entries
ESX Note: Supports both SQL (esx_vehicleshop.sql) and Lua formats. Ensure you have the latest version.
QBCore Note: Exports in QBShared.Vehicles format compatible with qb-core.
